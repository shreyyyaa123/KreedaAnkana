# KreedaAnkana

**KreedaAnkana** is an Android application designed for sports enthusiasts to discover, join, and organize local sports events and activities. The app provides a platform for users to connect, schedule games, and manage sports-related activities easily.

## Features

- **User Authentication:** Secure login/signup using Firebase authentication.
- **Event Discovery:** Find local sports events based on location and preferences.
- **Event Creation:** Users can create, manage, and invite others to sports events.
- **Notifications:** Receive updates about upcoming events and invitations.
- **Profile Management:** Personalize your profile and track your participation history.
- **Location-Based Services:** Integration with Google Maps to locate events.
